# android 下拉刷新、上拉加载相关的优秀第三方依赖包

* SmartRefreshLayout 下拉刷新、上拉加载、RefreshLayout、OverScroll，Android智能下拉刷新框架，支持越界回弹，具有极强的扩展性，集成了几十种炫酷的Header和 Footer。 
源码地址：[SmartRefreshLayout](https://github.com/scwang90/SmartRefreshLayout) 文档：[doc](https://github.com/scwang90/SmartRefreshLayout/blob/master/README-ch.md)

* PullRefreshLayout 下拉刷新，上拉加载，真实的回弹. 以实现任何你想的到的功能(例如:自动触发加载更多、二级刷新等)
源码地址：[PullRefreshLayout](https://github.com/genius158/PullRefreshLayout) 文档：[doc](https://github.com/genius158/PullRefreshLayout/blob/master/README.md)
