# Banner相关开源库

* [RecyclerViewBanner](https://github.com/Mr-XiaoLiang/RecyclerViewBanner)强烈推荐
	* 这是一个基于RecyclerView制作的Banner资源库
![](https://raw.githubusercontent.com/Mr-XiaoLiang/RecyclerViewBanner/master/video/demo.gif)

* [Banner](https://github.com/wenchaosong/Banner)
	* 轮播图控件，支持两端缩进，类似卡片，支持无限循环和多种主题，可以灵活设置轮播样式、动画、轮播和切换时间、位置、图片加载框架等！
![](https://github.com/wenchaosong/Banner/raw/master/pic/1.gif)
